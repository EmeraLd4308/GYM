# Features

Кожна папка — окремий домен продукту.

| Папка | Призначення |
| --- | --- |
| `auth/` | Форма входу |
| `workouts/` | Журнал, сесія тренування, списки, RPE, теги |
| `templates/` | Шаблони вправ |
| `stats/` | Графіки, streak, фільтри статистики |
| `calendar/` | Календар тренувань |
| `profile/` | Профіль, GL, досягнення, аватар |
| `dashboard/` | Головна, онбординг, дублювання дня |
